# Contributing to MailCraft AI

感谢你的贡献！

## 如何贡献

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的修改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 报告 Bug

请使用 GitHub Issues 报告 bug，并包含：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 截图（如有）

## 功能建议

欢迎提交功能建议！请描述：
- 功能用途
- 预期行为
- 为什么这个功能对项目有价值
